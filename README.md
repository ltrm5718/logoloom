## 🧭 What logoloom does

logoloom helps you create a logo, export it as SVG, and turn it into a full brand kit. It runs locally on your computer and works with MCP tools, so you can use it in a simple, private workflow. It is built for people who want clean brand assets without a long design process.

Use it to:

- make a logo from a short idea
- export clean SVG files
- create favicons and other small icon sizes
- build a basic brand kit from one design
- keep your files local on Windows
- connect with MCP-ready tools

## 💻 What you need

logoloom is made for Windows users who want a simple desktop setup.

You need:

- Windows 10 or Windows 11
- enough free disk space for the app and exports
- a mouse or trackpad
- an internet connection to get the release file

For best results, use a modern Windows PC with at least 8 GB of RAM. The app can run on less, but more memory helps when you work with larger SVG files or several brand assets.

## ✏️ How to use it

A simple way to work with logoloom is:

1. Start with a plain brand idea.
2. Add the name you want in the logo.
3. Try a few shape or icon options.
4. Choose a clean version that looks good at small sizes.
5. Export the logo as SVG.
6. Generate the rest of the brand kit from that base file.

Good logos stay simple. Keep the design clear so it works in a browser tab, on a website, and in print.

## 🎛 Common outputs

logoloom can help you make files like these:

- SVG logo

SVG is a good choice because it keeps edges sharp at any size. That matters when you use the same logo on a website, in a header, or on a business card.

## 🧩 Tips for better results

- Use short brand names when you can
- Keep shapes simple
- Pick one main color and one accent color
- Test the logo at small sizes
- Use strong contrast between text and background
- Export SVG first, then make other sizes from it
- Save your final files in one folder so they are easy to find

If a logo looks crowded, remove one element. Clean designs read better at a glance.

If the app opens but looks blank, close it and reopen it from the same folder you used the first time.
